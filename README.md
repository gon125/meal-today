# project-template-swift

## To Set Up
run scripts/.project-setup.sh
### What It Does
1. Linking `.swiftLint.yml`
2. Reassigning Git Hooks
3. Add Useful Code Snippets to Xcode
#### Git Hooks Contain
1. pre-commit

- Protect main branch from commits.
